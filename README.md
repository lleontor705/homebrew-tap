# Homebrew Tap for AITECH SOLUTIONS

This is the official Homebrew tap for AITECH SOLUTIONS packages.

## Installation

Install this tap:

```bash
brew tap lleontor705/homebrew-tap
```

## Available Formulae

### cortex

Persistent memory for AI coding agents — knowledge graph, importance scoring, vector search.

```bash
brew install lleontor705/homebrew-tap/cortex
```

### cortex-ia

AI Agent Ecosystem Configurator — persistent memory, SDD workflow, inter-agent messaging, multi-CLI orchestration.

```bash
brew install lleontor705/homebrew-tap/cortex-ia
```

## Usage

After installation, the binaries are available in your PATH:

```bash
cortex --help
cortex-ia --help
```

## Updating

To update all installed formulae:

```bash
brew upgrade
```

To update specific formula:

```bash
brew upgrade lleontor705/homebrew-tap/cortex
```

## Uninstallation

Remove a specific formula:

```bash
brew uninstall lleontor705/homebrew-tap/cortex
```

Untap this repository:

```bash
brew untap lleontor705/homebrew-tap
```
